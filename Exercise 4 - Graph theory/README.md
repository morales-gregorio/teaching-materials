# Graph theory with Python

In this folder the tutorial, exercises and solutions to the graph theory exercises for the Computational Neuroscience course in the University of Cologne. Re-used in the Introduction to Computational Neuroscience course at the Charles University (Prague)

Date of creation: June 2021

Author: Aitor Morales-Gregorio

Reviewed by: Agnes Korcsak-Gorzo, Sacha van Albada

### External data
The following external data is needed:
* `adjacency_marmoset.csv`

which is the adjacency matrix of connections inferred from retrograde injections [Majka et al 2020](https://www.nature.com/articles/s41467-020-14858-0). The adjacency matrix contains data for 55 cytoarchitectonic cortical areas (out of 116 areas described by the authors), with no unknown edges. The data is subject to a Creative Commons Attribution-ShareAlike 4.0 (CC-BY-SA) License.
